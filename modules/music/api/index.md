## Instalasi

Jalankan perintah di bawah di folder aplikasi:

```
mim app install api-music
```

## Endpoints

### {APIHOST}/music{?q,rpp=12,page=1}

### {APIHOST}/music/listen/{identity}

### {APIHOST}/music/album{?q,rpp=12,page=1}

### {APIHOST}/music/album/{id}
