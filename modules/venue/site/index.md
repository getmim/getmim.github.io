## Instalasi

Jalankan perintah di bawah di folder aplikasi:

```
mim app install site-venue
```

## Event Listener

1. venue:created (object $venue)
1. venue:deleted (object $venue)
1. venue:updated (object $venue)

## EndPoints

`GET /venue/read/(:slug)`
