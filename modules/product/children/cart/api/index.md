## Instalasi

Jalankan perintah di bawah di folder aplikasi:

```
mim app install api-cart
```

## Endpoints

`GET /cart`

`GET /cart/item`

`POST /cart/item {product,quantity}`

`DELETE /cart/item/(:id)`
