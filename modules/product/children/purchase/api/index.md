## Instalasi

Jalankan perintah di bawah di folder aplikasi:

```
mim app install api-purchase
```

## Endpoints

`/purchase?{page,rpp,status}`

`POST /purchase {items:[]}`

`GET /purchase/(:id)`

`DELETE /purchase/(:id)`

`GET APIHOST/purchase/(:id)/item`
